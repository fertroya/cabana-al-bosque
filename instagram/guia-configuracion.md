# Cabaña al Bosque — Guía para crear el perfil de Instagram

## 1. Crear la cuenta

1. Descargá **Instagram** en el celular o entrá a [instagram.com](https://www.instagram.com).
2. Registrate con un email o número de teléfono.
3. Elegí el nombre de usuario (sugerido): **`@cabanaalboscque`**
   - Alternativas: `@cabana.al.bosque`, `@cabanaalboscque.bariloche`
4. Tipo de cuenta: **Empresa** → categoría **Alojamiento / Casa de vacaciones**.

## 6. Después de crear la cuenta

1. Actualizá el link de Instagram en el sitio web si elegiste otro usuario.
2. En Instagram → Configuración → Cuenta → Cambiar a cuenta profesional → Alojamiento.
3. Conectá WhatsApp Business si usás el mismo número: +54 294 414-8173.
